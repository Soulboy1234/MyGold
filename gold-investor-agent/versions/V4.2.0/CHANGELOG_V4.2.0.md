# V4.2.0

更新时间：2026-03-11 07:35:00 +08:00

## 本版变更

- 优化了 `Agent1-基础` 的基础策略：提高普通再平衡缓冲，给非防守型卖出加上更明确的净收益门槛，并增加买入后的普通减仓冷却时间，减少评分边界附近的机械来回调仓。
- 优化了 `Agent2-短线选手` 的短线策略：把止盈型减仓和保护性减仓彻底分流，短线止盈现在要求更厚的利润空间；同时加入短冷却，避免刚买入后又为了薄利润匆忙减仓。
- 修复并优化了 `Agent2-短线选手` 的一个关键行为问题：当市场仍处于减仓或防守信号时，不再因为当前仓位略低于目标值而反向追回仓位。
- 进一步强化了 `Agent3-长线选手` 的长线特征：再次提高入场阈值、降低目标仓位档位、放大再平衡缓冲，并把普通长线减仓的冷却时间拉长，让它与 Agent1 的差异更明显。
- 三个 Agent 现在都保留了保护性减仓能力，但普通调仓卖出会更在乎手续费后的净收益质量，不再那么机械。
