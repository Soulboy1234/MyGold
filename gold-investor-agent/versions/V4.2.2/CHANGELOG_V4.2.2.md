## 本版变更

- 按最新要求恢复了 `Agent5-鬼才本人` 的挂单语义：挂单只看当前轮询快照价格，不再回看两次轮询之间的高频价格窗口。
- 这意味着如果价格在两次检测之间曾短暂触价、但到下一次 Agent 执行时又离开触发区间，那么这笔挂单会被视为“错过”，不会成交，继续保留在待执行列表中。
- 删除了上一版为 Agent5 挂单补加的中间窗口回看逻辑和相关辅助函数，避免保留无效复杂度。
- 保留当前 daemon 默认 5 分钟执行一次的设置，减少挂单因轮询过稀而错过的概率，但不改变“只看当前快照”的根本规则。
