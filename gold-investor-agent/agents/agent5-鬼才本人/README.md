# Agent5-鬼才本人

纯手动操作的黄金投资 Agent。

## 文件说明

- `agent.mjs`
  - Agent5 的执行入口
- `agent.json`
  - Agent 元数据
- `out/`
  - 持仓、交易、挂单和面板输出

## 风格定位

- 不使用自动交易策略
- 不依赖自动评分做买卖
- 平时自动运行只负责刷新估值、检查挂单、更新面板
- 真正的买卖由你通过面板手动提交

## 为什么没有 `strategy-config.json`

因为 Agent5 不是自动策略型 Agent，它没有固定的自动买卖参数可调。  
它当前支持的是：

- 手动买入
- 手动卖出
- 条件挂单买入
- 条件挂单卖出

如果以后你想给 Agent5 增加“半自动规则”，那时再给它增加单独的策略配置文件会更合理。
